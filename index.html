Renew Firestore Security Rules
After 30 days expire, follow these simple steps:
1 Go to console.firebase.google.com
2 Open project models-manager
3 From the menu select Firestore ← Rules
4 Delete all text and replace with:
rules_version = '2';
service cloud.firestore {
  match /databases/{db}/documents {
    match /{doc=**} {
      allow read, write: if true;
    }
  }
}
5 Click Publish — and done! ✅
