تجديد قواعد أمان Firestore
بعد انتهاء الـ 30 يوم، اتبع هذه الخطوات البسيطة:
1 اذهب إلى console.firebase.google.com
2 افتح مشروع models-manager
3 من القائمة اختر Firestore ← Rules
4 احذف كل النص واكتب هذا بدلاً منه:
rules_version = '2';
service cloud.firestore {
  match /databases/{db}/documents {
    match /{doc=**} {
      allow read, write: if true;
    }
  }
}
